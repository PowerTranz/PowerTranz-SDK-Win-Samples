Class PtzReceipt

PTZReceiptObject contains information that is highly recommended or required for a receipt. Additional information from the transaction object will be needed to generate a receipt, this object contains information that was generated by the card terminal, issuer, or a chip card itself, and should be included with any receipt printed or emailed
