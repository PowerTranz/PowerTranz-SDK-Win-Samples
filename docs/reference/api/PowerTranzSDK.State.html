Enum State

Possible states for the PowerTranz SDK State Machine. These states are optional information only regarding internal state and are not intended to be used for official responses.

Namespace: PowerTranzSDK
Assembly: PowerTranzSDK.dll
Syntax
public enum State

Fields

Name Description
Approved

Transaction was approved. Preparing response.
AwaitingAccountType

Waiting for the cardholder to select the account type (Credit or Debit)
AwaitingCard

Waiting for the cardholder to swipe or insert a card
AwaitingCashbackAmount

Waiting for the cardholder to enter a cashback amount. Only if cashback was requested.
AwaitingOnlinePin

Waiting for the cardholder to enter an online PIN. Only for debit with the appropriate service code. Offline PIN is handled by the terminal and is not controlled by the SDK.
AwaitingTip

Waiting for the cardholder to enter a tip amount. Only if tip was requested.
AwaitingTotalConfirmation

Waiting for the cardholder to confirm the total amount.
Canceled

Transaction was canceled. Preparing response and auto-reversal.
CardError

Card presented was not valid. Awaiting a card retry.
Connected

Terminal connection established
Connecting

Establishing connection with terminal (in progress)
Declined

Transaction was declined. Preparing response and optional auto-reversal.
DeviceArqcResponse

Terminal is processing ARQC response

DeviceContinuing

Terminal is continuing an EMV transaction after receiving amounts, total confirmation, and pin where required
DeviceEmvTrxnStarted

Terminal has started processing a new transaction
DeviceNfcTrxnStarted

Terminal has started processing a new NFC transaction
Disconnected

Not connected to terminal
Disconnecting

Disconnecting from terminal (in progress)
EmvTrxnRequested

A new transaction has been requested. This can be an EMV or MSR transaction, but not NFC.
Failed

Transaction failed with exception. Preparing response and auto-reversal.
Idle

SDK switches to Idle state after a transaction is completed
ProcessingOnline

Transaction is being sent to the Gateway for online processing
