Class PtzApi

Methods for communication to the PowerTranz Gateway Api

Inheritance
System.Object
PtzApi
Namespace: PowerTranzSDK
Assembly: PowerTranzSDK.dll
Syntax
public class PtzApi
Remarks

If there is an issue communicating with the Gateway a PtzApiException will be thrown. It contains an Http Response Code. 504 indicates that the Gateway timed out, and 503 indicates that there was some other communications issue.
