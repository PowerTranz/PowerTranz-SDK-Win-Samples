Namespace PowerTranzSDK.CardTerminals.Miura

Classes

PTZMiuraTerminal

An instance of the Miura terminal

Delegates

CardStateChanged

Indicates an internal card state changed (state machine transition)

DeviceTextCompletion

Internal delegate used by the SDK

DidConnectTerminal

This delegate function is called when a communication channel has been opened with the credit card terminal.

DidDisconnectTerminal

This delegate function is called when the communication channel to the credit card terminal has been closed.

DidFail

This delegate function is called when an error has been detected when performing a requested task.

DidFailToConnectTerminal

This delegate function is called when the connection to the Terminal hardware fails to be created.

DidFailWithReversal

This method is called when a reversal is performed on a transaction with an error or exception

DidFinish

This delegate function is called after the terminal has generated the Transaction Certificate or Application Authentication Cryptogram and no more online requests are pending.

DidFinishWithReversal

This method is called when a reversal is performed

DidReceiveCardEvent

This delegate method is called when the terminal receives a card event. It does not return the card data and therefore is called in the event that the card data is not going to be returned for whatever reason.For instance, a card swipe does not return data if the card is IC capable and was swiped.

DidReceiveResponse

This method is used to convey the response body that was sent from the server

DidRequestDevicePromptText

This delegate method is called when the terminal would like text displayed by the user. If the terminal is capable of displaying text (terminalCanDisplayText) and this method is implemented, then the terminal should be sent a text string to display to the user. This can be done via the completion handler, with a NIL string to have the terminal display default text.

OnBatteryLow

This delegate method is called when the terminal sends a low battery status message.

OnBatteryPercentage

This delegate method is called when the terminal sends a battery percentage event;

PromptCanceled

This delegate function is called when a user presses cancel on the keypad.

SdkStateChanged

Indicates an internal SDK state change occurred (state machine transition)

TerminalDidReceiveKeypadInput

This method is called when a key is pressed on a supported device (Miura). For EMV / PIN security, the key press will not be relayed to the delegate and therefore this message is only useful if the application cares about keypad presses for whatever reason.

WillSendRequest

Transaction request details about to be sent online. For development purposes only.
